## Dynamic Badges

Dynamic badges automatically update based on your project’s real-time data — no manual changes needed. They are ideal for showing live stats directly in your README

🏷️ Version
| Platform       | Badge   | Markdown                                                              |
| -------------- | ------- | --------------------------------------------------------------------- |
| GitHub Release | Release | `![Release](https://img.shields.io/github/v/release/<user>/<repo>)` |
| NPM            | NPM     | `![NPM](https://img.shields.io/npm/v/<package>)`                    |
| RubyGems       | Gem     | `![Gem](https://img.shields.io/gem/v/<gem>)`                        |
| Flathub        | Flathub | `![Flathub](https://img.shields.io/flathub/v/<package>)`            |
| PyPI           | PyPI    | `![PyPI](https://img.shields.io/pypi/v/<package>)`                  |
