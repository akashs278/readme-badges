## Dynamic Badges

Dynamic badges automatically update based on your project’s real-time data — no manual changes needed. They are ideal for showing live stats directly in your README

🏷️ Version
| Platform       | Preview | URL Syntax                                              |
| -------------- | ------- | ------------------------------------------------------- |
| GitHub Release | Release | `https://img.shields.io/github/v/release/<user>/<repo>` |
| NPM            | NPM     | `https://img.shields.io/npm/v/<package>`                |
| RubyGems       | Gem     | `https://img.shields.io/gem/v/<gem>`                    |
| Flathub        | Flathub | `https://img.shields.io/flathub/v/<package>`            |
| PyPI           | PyPI    | `https://img.shields.io/pypi/v/<package>`               |
